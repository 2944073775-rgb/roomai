# 房源图片美化工具 - 技术方案

## 产品目标
帮助房产中介批量美化房源图片（提亮、去噪、校正），操作简单，扫码即可上传，15张/批次，无水印。

## 技术架构

### 前端
- 单页 HTML 应用
- 响应式设计（电脑端操作 + 手机端扫码上传）
- 无需后端，纯前端处理（AI 修图除外）

### 核心功能
1. **二维码生成** - 手机扫码上传
2. **批量上传** - 最多15张，支持拖拽
3. **图片预览** - 网格展示，可单选/全选
4. **批量美化** - 一键处理所有图片
5. **滑动对比** - 真正的拖动滑块对比原图/效果图
6. **批量下载** - ZIP打包下载，无水印
7. **美化参数可调** - 亮度/对比度/饱和度/锐化/降噪滑块
8. **AI 修图** - 去杂物/局部修改（接 DeepAI Inpainting API）

### 图像处理方案

#### 方案A：客户端处理（当前版本）
使用 Canvas API 做基础处理：
- 亮度/对比度调整
- 色温校正
- 锐化
- 透视校正（需要手动标注或自动边缘检测）

限制：去噪能力有限，无 AI 增强

#### 方案B：接第三方 API（生产版本）
推荐：
- **图像增强**: DeepAI、Clarifai、Baseten
- **去噪**: DnAPI、remove.bg(消人物专用)
- **透视校正**: API4AI、Zygo

### 二维码方案
- 使用 qrcode.js 生成二维码
- 扫码后跳转至 /upload 页面
- 图片通过 Base64 传输或 WebSocket

## AI 修图功能（2026-04-17 新增）

### 使用方式
1. 点击"AI 修图"模式卡片
2. 在遮罩画布上涂抹要修掉的位置（红色区域）
3. 输入描述文字，如"去掉地上的垃圾"
4. 点击"AI 修图"处理

### API 配置
- 使用 **OpenAI DALL-E 2**
- 支持图像编辑/修图（带遮罩）
- API: `/v1/images/edits`
- 注册：https://platform.openai.com/

### 遮罩绘制
- 支持鼠标和触摸操作
- 细笔（12px）/ 粗笔（35px）
- 可清空重画
